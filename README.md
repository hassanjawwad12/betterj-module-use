# betterj-module-use
Using the created betterj module

Here is the link to the module code and instructions for creating it: [GitHub Repo](https://github.com/hassanjawwad12/betterj/tree/main)

# steps to follow 
* Initialize the go module in new directory `go mod init <your-module-name>`
* Download the module by using `go get github.com/hassanjawwad12/betterj` 
* Create a main.go file and use this import `"github.com/hassanjawwad12/betterj"`
* Run the `main.go` file after implementation
